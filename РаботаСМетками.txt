Просмотр имеющихся меток:
git tag

Поиск метки по шаблону:
git tag -l 'v1.4.2.*'

Создание меток:
метки: аннотированные(указатель на коммит) и легковесные(похоже на ветку)

Аннотированные метки хранятся как полноценные объекты.
Имеют контрольную сумму, имя поставившего метку, эл.почту,
дату, комментарий, подпись.

Создание аннотированной метки:
git tag -a v1.4 -m 'my version 1.4'

Посмотреть информацию:
git show v1.4	# странно работает, не так как описано

Удаление метки:
git tag -d tag_name

Перемещение тега:
git tag -f tag_name

Легковесная метка:
git tag v1.4-lw

Верификация меток
git tag -v [имя метки]   # Как получить открытй ключ автора подписи?

Выставление меток позже:
git tag -a v1.2 -m 'version 1.2' 9fceb02
# в конце указана контрольная сумма или ее часть

Обмен метками: 
git push не отправляем метки
git push origin [имя метки]
git push origin --tags 	# отправить все метки
